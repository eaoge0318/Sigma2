## 核心术语映射

| 制程术语 | 通用术语 | 适用场景示例 |

## 代码中的术语

### Python代码

**保持不变**（技术术语）：
- `parameter` - 变量名保持
- `get_parameter_list` - 函数名保持
- `required_params` - 技术术语保持

**修改显示文本**（用户可见）：
```python
# 修改前
display_name = "参数列表"
description = "获取所有参数"

# 修改后
display_name = "字段列表"
description = "获取所有字段"
```

### JavaScript代码

**变量名不变**：
```javascript
const params = data.parameters;  // 技术变量名保持
```

**显示文本修改**：
```javascript
// 修改前
document.querySelector('.label').textContent = '参数数:';

// 修改后
document.querySelector('.label').textContent = '字段数:';
```

## 总结

### 核心原则

1. **前端UI**：完全使用通用术语（字段、指标、影响因素）
2. **后端代码**：技术变量名保持不变，display_name使用通用术语
3. **文档示例**：优先使用通用示例（价格、面积），避免制程特定示例

### 通用性优势

- ✅ 适用于各行业数据分析
- ✅ 用户无需了解特定领域
- ✅ 降低学习门槛
- ✅ 提高系统复用性
